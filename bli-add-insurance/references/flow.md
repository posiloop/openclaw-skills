# BLI 加保作業流程

## Canonical flow

1. 開啟首頁  
   `https://edesk.bli.gov.tw/me/#/home`
2. 點擊 `投保單位網路申報及查詢作業`
3. 關閉登入前公告，點擊 `Close`
4. 選擇登入方式 `自然人憑證`
5. 先直接點擊 `檢測讀卡機`
6. 不需要先停下來確認是否跳出原生提示，直接繼續填入登入資訊
   - 身分證號
   - 保險證號
   - 自然人憑證密碼（PIN）
7. 點擊 `送出`
9. 等待約 10 秒，再判斷是否成功跳轉到  
   `https://edesk.bli.gov.tw/me/#/cpa/home`
10. 若等待期間畫面看起來沒有變化，不要立刻視為失敗，先等滿約 10 秒
11. 若等滿後仍未前進，再檢查是否有 Chrome 或 macOS 的自然人憑證、讀卡機、外部程式確認視窗
12. 若有原生提示，請使用者手動允許後，再從目前頁面狀態繼續
13. 若出現 `歡迎來到新手導覽`，點擊 `略過導覽`
14. 直接訪問 `https://edesk.bli.gov.tw/me/#/cpa/apply/sgl-declare/create`（跳過功能選單）
15. 第一步維持預設 `勞(就,災)保/勞退`
16. 點擊 `下一步`
15. 進入加保資料頁後，以互動方式逐步蒐集資料，而不是一次索取全部欄位

## Data-entry interview order

1. 先詢問加保人：
   - 身分證字號
   - 姓名
   - 出生日期
2. 完全忽略性別欄位。不讀取、不點擊、不設定、不驗證。申報不需要此欄位即可成功。
3. 出生日期一律轉成民國格式 `YYYMMDD`
   - 例：民國 76 年 3 月 11 日 → `0760311`
4. 打開 `加保日期` 下拉選單
4. 將畫面中所有實際選項列給使用者
5. 在選項最後額外加上一個 `夜間到職`
6. 詢問使用者選擇哪個加保日期
7. 詢問特殊身份別，提供以下選項：
   - `0：無`
   - `1：部分工時人員`
   - `3：董事（實際從事勞動）`
   - `4：受委任工作者`
   - `5：雇主（單位負責人）`
   - `A：功法救助及部分工時人員`
   - `B：功法救助`
   - `S：建教生及部分工時人員`
   - `T：建教生`
8. 若回答 `無`，可略過該欄位
9. 詢問月薪資總額
10. 詢問是否填寫 `個人自願提繳率`
    - 要填就索取數字
    - 不填就略過
11. 詢問 `勞退提繳日期` 是否與（勞/退/災）加保日不同
    - 若不同，索取日期並轉成 `YYYMMDD`
    - 若相同，略過
12. 詢問是否同時加保健保
    - 若是，索取 `合於健保投保日期`，格式 `YYYMMDD`
    - 若否，取消勾選健保本人
    - 一般情況不要額外追問 `合於健保投保條件`，沿用頁面既有或預設值即可；只有在使用者要改，或畫面值與情境明顯不符時才處理
13. 詢問是否申報眷屬加保
14. 若要申報眷屬，逐筆詢問：
    - 姓名
    - 身分證號
    - 出生日期（`YYYMMDD`）
    - 稱謂代號
    - 合於健保投保條件
    - 合於健保投保日期（`YYYMMDD`）
15. 每位眷屬確認完成後：
    - 點 `申報眷屬加保`
    - 等新區塊出現
    - 填完後按 `新增`
16. 若還有下一位眷屬，重複同樣流程
17. 最後填入 `自然人憑證密碼`
18. 再次確認資料無誤後，才按 `申報`

## Dependent option sets

### 稱謂代號

- 配偶
- 父母
- 子女
- 祖父母
- 孫子女
- 外祖父母
- 外孫子女
- 曾祖父母
- 外曾祖父母

### 合於健保投保條件

- 隨同被保險人加保
- 喪失被保險人身份
- 結婚
- 收養
- 新增嬰兒
- 更換所依附之被保險人
- 在學就讀且無職業（在學無業）
- 受禁治產宣告尚未撤銷（禁治產宣告）
- 領有殘障手冊而不能自謀生活（殘障不能謀生）
- 罹患重大傷病且無職業（重傷病無業）
- 應屆畢業或服兵役退伍且無職業

## Fragile points

### 讀卡機檢測

這一步應先照常執行點擊，之後直接繼續輸入登入資料並送出。只有在送出後流程明確卡住時，才需要回頭檢查是否有瀏覽器或作業系統原生 UI 需要使用者手動確認。

### 登入後導覽

`歡迎來到新手導覽` 不一定每次都出現。若沒有出現，不要因此卡住流程。

### 功能選單（已不使用）

登入後改為直接訪問加保作業 URL，不再透過功能選單導航。保留此段僅供參考。

### 加保日期

不要預設日期內容。要先打開畫面上的下拉選單，讀取實際選項，再列給使用者選。

## Recommended stopping points

- 完成登入後
- 進入加保作業頁後
- 申報前最後確認時
