# 三檔 → 現金利潤表 映射規則

適用來源檔：
1. `星夢115年零用金明細.xlsx`（petty cash — sheets：便當 / 財務 / 工程費用 / 陳楟蓁 / 姜建良）
2. `星夢匯款明細表.xlsx`（remit — sheets：115年營收收支明細 / 2026年匯款明細 / 2026陽信金流 / 2026年1–3月銀行明細）
3. `星夢餐飲區零用金.xlsx`（F&B petty cash — sheets：餐飲20263月前 / 餐飲2025 / 餐飲區）

目標格式見 `stardream-115-feb-sample.pdf`。

## 1. 檔案角色

| 檔案 / Sheet | 角色 |
|---|---|
| 2026年匯款明細 | **主要支出來源**（所有對廠商的撥款） |
| 2026陽信金流 / 2026年X月銀行明細 | 僅作對帳驗算，**不計入金額** |
| 115年營收收支明細 | 營收 + 少量零星收支 |
| 便當 / 財務 / 工程費用 / 陳楟蓁 | 零用金現金收支 |
| 姜建良 | **忽略**（目前空白） |
| 餐飲20263月前 / 餐飲區 / 餐飲2025 | 餐飲區零用金 |

## 2. 欄位辨識

- 零用金格式：`年 月 日 摘要 收入 支出 餘額 發票號碼 憑証 備註`（有時多 `類別` 或 `歸屬`）
- 匯款明細：`預計日期 摘要 金額 手續費 廠商 銀行 … 實際撥款金額 實際撥款日 憑証`

## 3. 期間與認列

- 期間：以 `年=2026, 月=M` 為 primary filter
- 認列日期：
  - 匯款明細 → 以**實際撥款日**為準
  - 零用金表 → 以**年/月/日**為準
- 認列口徑（覆蓋上一條）：
  - 以下項目以**當期認列**：天使星夢費用、薪資、勞健退、運管費用
  - 其他項目以**付款時間認列**
- 天使星夢的「當期」= 摘要中「X 月收益分潤 …」的 X 月

## 4. 排除規則

以下資料不計入現金利潤表：

- 零用金入金（如「零用金 41,000」入）：屬內部資金移轉，非營業收入
- 結轉列（只有餘額欄位、無收入/支出金額）
- `姜建良` sheet 全部
- 銀行明細 sheets（僅對帳）

## 9. 防重複計入

- 匯款明細 vs 銀行明細：**匯款明細為準**；銀行明細對帳驗算用、不計入
- 注意匯款明細與財務零用金表的內容通常不重複（匯款是廠商撥款、零用金是小額現金），但若同一筆出現在兩處，以匯款明細為準

## 9.5 未分類項目預設歸類（fallback）

當一筆資料無法命中 §5、§6、§7、§8 任一規則時：

- **收入面** 未分類 → 歸類為「**其他**」
- **支出面** 未分類 → 歸類為「**營業支出 > 雜支**」

fallback 僅在 §12 衝突處理優先順序用盡後使用，不覆蓋既有明確規則。

## 10. 輸出結構（維持 SKILL.md 既定）

- 合計 = 總營收 − 總支出（可為負，以括號呈現，如 `(568,362)`）
- 百分比（利潤比）= 該項金額 / 同欄總營收
- 數字千分位逗號；空值 `-` 或留白
- 三欄一組：`當月利潤 / 利潤比`，最右側追加 `當年利潤 / 利潤比`（YTD）

## 11. 決策歷程（來自 2026-04-23 對話）

| # | 議題 | 決議 |
|---|---|---|
| 1 | 便當表歸屬 | 人事成本 |
| 2 | 陳楟蓁「歸屬」欄 | 採用為一級分類線索 |
| 3 | 碳粉匣、墨水匣 | 營業支出 > 雜支 |
| 4 | 補充保費 | 勞健保 |
| 5 | 匯款 vs 銀行 | 匯款為準，銀行僅對帳 |
| 6 | 零用金入金 | 排除，不列為收入 |
| 7 | 姜建良 sheet | 忽略 |
| 8 | 天使星夢來源 | 廠商 == `天使星夢渡假會館`，當期認列 |
| A | 「當期認列」定義 | **費用所屬月**（例：12 月薪資認列在 12 月，不論何時撥款） |
| M | 未分類收入 | fallback → 收入「其他」 |
| N | 未分類支出 | fallback → 支出「營業支出 > 雜支」 |
| B | 金額 vs 實際撥款金額 | 以 **實際撥款金額** 為準（含手續費） |
| C | 陳楟蓁摘要 vs 歸屬 | **摘要關鍵字優先**（退佣→退佣&勞報單&發票 勝過 歸屬→營業支出） |
| D | 收入欄「還沒入支出」 | 列入收入或從支出沖抵（依實際性質判斷） |
| E | 信託差額（付給自家信託專戶） | **排除**，屬內部資金移轉 |
| F | 預計撥款但未實際撥款 | **等實際撥款才認列** |
| G | 森山收入 vs 森山支出 | **以收入 / 支出欄位自動判斷**（摘要含「森山」，在收入欄→森山；在支出欄→森山支出） |
| H | 陳楟蓁歸屬值 | 運營→營業支出；餐飲→餐飲區用品；園區→園區支出；人事→人事成本 |
| I | 餐飲區 sheets 關係 | 三個 sheet 彼此**不重疊**，2026 年報表使用 `餐飲20263月前`、`餐飲區`；`餐飲2025` 屬 2025 年資料 |
| J | 期初餘額列 | 排除（如「114 年餘額」「114 年便當餘額」只有餘額欄位的列） |
| K | 凱基轉陽信匯費 / 內部轉帳 | 跟著主項目一起歸類（以實際撥款金額已含手續費處理，見 B） |
| L | 補充保費期別 | 依 A：以費用所屬月認列（「11 月薪資補充保費」→ 認列 11 月） |

## 12. 關鍵衝突處理優先順序

當一筆資料可被多條規則命中時，優先順序如下：
1. 排除規則（§4）
2. 固定廠商規則（§6.1，例如天使星夢）
3. 摘要關鍵字（§6.2、§7、§8）
4. 歸屬欄（陳楟蓁的 §6.4 / §H）
5. Sheet 來源（如工程費用 → 園區支出 > 維修）

### 12.1 摘要關鍵字內部的優先順序

當同一筆摘要同時命中多個關鍵字（例如「12月份淘寶進貨(周邊+聖誕節+過年佈置)」同時含「淘寶進貨」和「佈置」），以「主要採購/費用性質」優先，次要修飾詞退讓：

- 「淘寶 / 掏寶」優先於「佈置 / 糖果 / 春聯 / 紅包袋」等展場裝飾詞 → 掏寶進貨
- 「薪資 / 獎金 / PT」優先於「便當 / 員工」 → 人事成本
- 「退佣 / 勞報單」優先於歸屬欄 → 退佣&勞報單&發票（§11 決策 C 延伸）

簡單判斷：摘要開頭出現的詞、代表資金流向對象的詞，優先級高於附註括號內的描述詞。
